>> <h3> Step I - HOW WORKS </h3>

<p> After downloading clone the repository run the file country.sh </p>

<p> It asks you to enter the name of some country. And returns a capital. This answer is by querying wikipedia  </p>

>> <h3> Help command </h3>
<p> type: </p>

`sh country.sh --ajuda`
